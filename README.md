# Gerador e Validador de CPF/CNPJ

## Descrição

Este projeto oferece ferramentas para geração e validação de CPFs e CNPJs. Ele é útil para desenvolvedores que precisam de dados válidos para testes.

## Estrutura do Projeto

- **`gerador_de_cpf_cnpj.ipynb`**: Notebook com a lógica para geração de CPFs e CNPJs válidos.
- **`validador.ipynb`**: Notebook que valida CPFs e CNPJs com base em algoritmos oficiais.
- **`base/`**: Pasta que pode conter dados de suporte para o projeto (detalhes dependem do conteúdo específico).
- **`export/`**: Pasta destinada para saída/exportação de arquivos ou resultados.

## Funcionalidades

1. **Geração de CPFs e CNPJs**:
   - Gera números válidos conforme as regras do dígito verificador.
   - Útil para criação de dados fictícios em sistemas de teste.

2. **Validação de CPFs e CNPJs**:
   - Verifica a validade de um número de CPF ou CNPJ.
   - Ajuda a garantir a conformidade com as regras brasileiras de validação.
